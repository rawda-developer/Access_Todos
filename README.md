## üèÅ Getting Started <a name = "getting_started"></a>

### Prerequisites

```
node -v
npm -v
```

### Installing

1. Clone the repository

```
git clone https://github.com/rawdaymohamed/Hot-Todo
cd Hot-Todo
```

2. For the frontend

```
cd frontend
npm install
npm start
```

3. For the backend

```
cd ..
cd backend
npm install
npm start
```
